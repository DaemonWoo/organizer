# Organizer for Cluttered Folders

Familiar situation?
- Has your **Downloads** folder exploded to thousands of files?
-	Can’t find installers among screenshots?
-	Documents get lost among images and archives?
-	Manual sorting takes hours?

✅ Solution:
**Organizer** scans your folder, detects file types by extension, and moves them into the appropriate category.

## ✨ Features
* **Interactive Menu:** Choose preset folders (Downloads, Desktop, Documents) or provide a custom path.
* **Smart Sorting:** Automatically categorizes files into target folders in your Home directory. (e.g. Documents, Music, Archive)
* **Safe Moving:** Never overwrites existing files. Automatically renames duplicates (e.g., `file (1).jpg`).
* **Dry Run / Summary:** Shows a preview of what will be moved before making any actual changes.
